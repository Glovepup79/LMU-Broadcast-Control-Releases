## Auto-Update

Checks for new releases on startup and notifies you when an update is available.

## Live Timing Grid

Real-time positions, sectors, gaps, energy, penalties, and race status.
Includes close racing and slow car highlighting.

## LMU Overlay & Broadcast Control

- Standings tower control.
- Driver and class filtering.
- Safety Car and Code80 toggles.
- Starting grid display.

## Camera & Replay

- One-click driver focus.
- Camera family switching.
- Automatic replay triggers for incidents.

## Beta Overlay

Standalone web-based overlay.

- Live timing tower.
- Qualifying information panel.
- 1v1 battle overlay.
- Session and mid-race results panels.
- Track map and starting grid carousel.

## Remote PC Support

Connect to up to 2 LMU PCs.
Uses REST and WebSocket (ports 6397 / 6398).
